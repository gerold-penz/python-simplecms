Simple Python CMS¶

Content Management System - Made Simple

Mein Ziel ist es, ein Content Management System (CMS) zu programmieren, das möglichst einfach (zumindest in meinen Augen) bedienbar ist. Bitte nicht falsch verstehen: Es ist nicht Teil des Ziels, ein einfach zu installierendes CMS zu erstellen. Die Installation hat es in sich. Das Ergebnis wird aber ein sehr einfach bedienbares CMS sein.

Technik¶

Als begeisterter Python-Programmierer werde ich als Web-Framework CherryPy einsetzen. Als Vorlagensprache habe ich mich für Mako entschieden.
